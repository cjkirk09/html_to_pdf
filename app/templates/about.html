This page demonstrates the different types of link tags you can define.

Typically a css link tag looks like this:

<link href="/some/path/to/file.css" rel="stylesheet" type="text/css">

But there is another parameter you can define in the link tag. media

If you define media="screen", then the page will use the css defined in that file for screen viewing only (even if you have defined some @media print css in the file).
But if you define media="print", then the page will use the css defined in that file for print viewing (even if you have defined some @media screen css in the file).
