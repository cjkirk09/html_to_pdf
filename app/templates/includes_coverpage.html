Includes Coverpage

One area where there is a clear winner is in adding coverpages to the generated pdf.

It is possible that PhantomJS has this ability, but I have not discovered it yet. So if it does have it, then it is not very well documented or used.


But it comes built into the documentation of WkHtmlToPdf and is easy to use.

I made it so that if you add ?cp=t to your url when request that WkHtmlToPdf turn it into pdf, it will add a coverpage.
